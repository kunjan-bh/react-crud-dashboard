# React CRUD Dashboard

A simple React CRUD Dashboard built using a modern UI library. This project allows users to:

1. View all posts in a table

2. Create a new post

3. View details of a specific post

4. Update an existing post

5. Delete a post

All operations use the JSONPlaceholder REST API for demonstration.

# Technologies Used

1. React

2. UI Library: Material UI

3. Axios for API requests

4. Vite / Create React App 

# Installation
```bash
git clone https://github.com/kunjan-bh/react-crud-dashboard
cd crud-dashboard
npm install
npm run dev
```
